Evidence Map User Guide
This is an interative search interface for PubMed articles indexed by PIO elements. We provide multiple search and result filtering methods.
  • Plain Search
    • The system supports searching by PubMed ID, key words or question. E.g. PMID: 32037483; Keyword: COVID-19; Clinical Question: Do chloroquine or hydroxychloroquine improve outcomes in COVID-19 infection?

  • Advanced Search

    • "Advanced Search" provides a Boolean search bulider where logical operators (AND, OR) can be used in addition to the keywords.

    AND: The AND operator tells the search engine to return only documents with all the keywords you entered. This operator narrows the search and returns fewer search results.

    OR: The OR operator tells the search engine to return documents if they contain one or more keywords.

    Example:

    The logic between PIO types is "AND" that cannot be modified. The above example defined a search query: (Participant = "COVID" AND "T2DM") AND (Intervention = "Hydroxychloroquine" OR "Remdesivir") AND (Outcome = "Intubation")

  • Result Filtering

    • You can refine your results by using the "PIO Filter" on the left of the results page. "PIO Filter" narrows your results based on the selected PIO elements. For example, you can view PubMed articles only including the selected PIO elements.

  • PIO Elements Display and Download
    • PIO elements are tagged in the abstract of PubMed article and you can download the parsing results in JSON format by clicking the "JSON" button in the right corner. The page will be redirected to the original article after clicking the PubMed ID.
